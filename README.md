# Slider after to before

## Props
```
slider: HTMLElement ,
classSlide: string,
classSlider: {
  [key: number]: string
},
textSlider: {
  [key: number]: string
},
```

- classSlide класс влоденый в slider HTML элементов
- classSlider класс деваядера обозначается где key его индекс
- textSlider текст деваядера обозначается где key его индекс
